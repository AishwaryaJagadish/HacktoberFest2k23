REQUIREMENTS:
Pygame needs to be installed.
To install pygame:
1.Open command prompt(as administrator)
2. Make sure you have working internet connection.
3.Type "pip install pygame"
Done! now execute the main.py

CONTROLS:
Use A for Left, D for Right and SPACEBAR to fire gun.
------------------------------------------------------
DISCLAIMER:
This game is strictly for entertainment purpose only.
------------------------------------------------------
You'll be required to enter number of enemies in range of [1,50] and in integer value only
if it's not an integer value game will take the default value of 6 and execute the game.
